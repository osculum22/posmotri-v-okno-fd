# ПРОЕКТНАЯ РАБОТА "ПОСМОТРИ В ОКНО"

Данная проектная работа позволяет тренировать **навык работы 
с уже частично написанным кодом**, в отличие от предыдущей работы "ОНО ТЕБЕ НАДО",
в которой код создавался почти с нуля.

Т.к. в реальных проектах часто приходится стилизовать готовые 
плагины, менять темы, созданные другими разработчикамми, дописывать
стили в уже готовые модули, данная проектная работа как раз и необходима 
для знакомства с порядком действий в таких случаях.

# Посмотри в окно

Фронтенд-приложение для поиска и просмотра видео о городах и времени суток. Проект представляет собой интерфейс с формой поиска, списком видео-карточек и плеером.

## Системные требования

Для запуска проекта вам понадобится любой современный браузер и локальный веб-сервер для разработки.

### Требования

-   **Браузер:** Google Chrome, Mozilla Firefox, Safari, Edge (любая актуальная версия)
-   **Локальный сервер**
